# 🏡 Housing Price Prediction using Machine Learning

A Machine Learning web application that predicts house prices based on property features such as area, number of bedrooms, bathrooms, and other housing attributes.

---

# 🚀 Live Demo
https://housing-price-ml-kgdunabyzyd2ex8rbmusjs.streamlit.app/



---

# 📌 Project Overview

This project uses a Machine Learning regression model to estimate housing prices from user-provided property details. It provides an easy-to-use web interface built with Streamlit, allowing users to explore predictions instantly.

---

# ✨ Features

* 🏠 Predict house prices using Machine Learning
* 📊 User-friendly Streamlit interface
* ⚡ Instant price prediction
* 📈 Clean and responsive dashboard
* 🤖 Trained regression model using Scikit-learn

# ⚙️ Installation

Clone the repository

```bash
git clone https://github.com/YOUR_USERNAME/Housing-Price-Prediction.git
```

Move into the project

```bash
cd Housing-Price-Prediction
```

Create a virtual environment

```bash
python -m venv .venv
```

Activate it

**Windows**

```bash
.venv\Scripts\activate
```

Install dependencies

```bash
pip install -r requirements.txt
```

Run the application

```bash
streamlit run app/app.py
```
